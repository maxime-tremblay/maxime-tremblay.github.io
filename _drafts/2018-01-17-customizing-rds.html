--- layout: post title: Customizing RDS date: '2018-01-17T16:34:00.000-05:00' author: Maxime Tremblay tags: modified_time: '2018-01-17T16:34:00.724-05:00' thumbnail: https://3.bp.blogspot.com/-u8Wlvk_O4C0/Wl-_44LUptI/AAAAAAAABj0/sXTzm6a0q78DTDB3k-8Itjy65Ib3o7JOwCLcBGAs/s72-c/tabs-simple.png blogger_id: tag:blogger.com,1999:blog-267713902572011003.post-4640608284701620914 blogger_orig_url: https://max-tremblay.blogspot.com/2018/01/customizing-rds.html --- Tabs and Region Display Selector can be used to improve navigation, flow and usability of your pages.

Using the Tab Style Template Option, we're able to choose between two display style for the Tabs

  • Simple: A very simplistic tab UI.


  • Pill: Displays tabs in a pill container.


There's also the Layout Template Option that we can use to stretch tabs to fill to the width of the tabs container.

But these Template Options don't exist for the Region Display Selector regions.

The way that Template Options works is that they are adding CSS classes to the different elements.
Let's do the same for "RDS as Pill" and "RDS Fill Lables".

/* ********************** */
/* RDS as Pill            */
/* ********************** */
/* Note: Color attributes will need to be adapted according to your Theme Roller Style */
.apex-rds--pill .apex-rds {
    background-color: #f9f9f9;
    border: 1px solid rgba(0,0,0,.1);
    background-clip: padding-box;
    box-shadow: 0 1px 2px rgba(0,0,0,.05);
    border-radius: 2px;
    margin-bottom: 8px;
}

.apex-rds--pill .apex-rds-item a {
    border-left: 1px solid #E0E0E0;
    color: #404040;
}

.apex-rds--pill .apex-rds .apex-rds-selected a {
    color: #404040;
} 

.apex-rds--pill .apex-rds-item.apex-rds-selected a,
.apex-rds--pill .apex-rds-item:hover a {
    background-color: #ffffff;
}

.apex-rds--pill .apex-rds .apex-rds-selected span {
    box-shadow: none;
}

/* ********************** */
/* RDS Fill Labels        */
/* ********************** */
.apex-rds--fillLabels .apex-rds {
    display: table;
    table-layout: fixed;
    border-collapse: collapse;
    width: 100%;
}

.apex-rds--fillLabels .apex-rds-item {
    display: table-cell;
}

.apex-rds--fillLabels a {
    display: block;
}

Now we can add "apex-rds--fillLabels" and/or "apex-rds--pill" to the RDS region's CSS Classes (under the Layout section).
